## 一、安装

1. 确保电脑上安装有 node 环境且 npm 命令可用。
2. 新建文件夹，使用`npm init`新建项目
3. 使用`npm install -g webpack`安装 webpack 命令
4. 项目文件夹下执行`npm install webpack webpack-cli --save-dev`进行脚手架安装
